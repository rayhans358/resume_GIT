What is GIT? Manfaat GIT Kapan di butuhkan
  1. GIT adalah salah satu version control system yang diciptakan oleh Linus Torvalds
  2. GIT mencatat setiap perubahan file di code atau sourcode kita
  1. Bisa menyimpan seluruh versi source code
  2. Berkolaborasi membangun sistem jadi mudah
  3. Bisa berkontribusi di proyek open source
  4. Memudahkan dalam tracking perubahan
  5. Memahami cara deploy modern
  • Saat ingin membangun sistem dengan berkolaborasi
  • Membuat versi dari sistem
  • Backup sour code
  • Berkolaborasi dengan programmer dunia
